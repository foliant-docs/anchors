# Special symbols

Accepted:

<anchor>!important!</anchor>
<anchor>_anchor_</anchor>
<anchor>section(1)</anchor>
<anchor>section/1/</anchor>
<anchor>anchor$1$</anchor>
<anchor>about:info</anchor>
<anchor>test'1';</anchor>
<anchor>якорь</anchor>
<anchor>👀</anchor>


Not accepted:

<anchor>foo[bar</anchor>
<anchor>foo]bar</anchor>
<anchor>foo>bar</anchor>
<anchor>foo\bar</anchor>
<anchor>foo"bar</anchor>
<anchor>foo bar</anchor>
